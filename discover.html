Question-1: What are the different ways to select an element in the Dom ?

There are several ways to select an elements from Dom. firstly getElementById() best for selecting a element then getElementByClassName() select by class name it returns an HTML collection (like an array) next one is getElementsByTagName() next is querySelector() and then querySelectorAll() for all matching Elements.

Question-2:What is the difference between innerText, innerHTML and textContent ?

innerText : it returns only visible Text. innerHTML : Gets or sets all HTML inside an element not just text. textContent : it Returns all text includes hidden elements also.

Question-3: What is the event Delegation in the Dom ?

It allows you to handle events more efficiently by adding a single event listener to a parent element instead of multiple listeners on individual child elements. No need to add or remove listeners manually and best uses of when multiple similar elements exist inside a container.

Question-4: What is event bubbling in the Dom ?

where an event triggered on a child element propagates (bubbles up) to its parent elements until it reaches the root.
